
# SuperChase Agents (Quick Start)

1) Save your OpenAI API key: `export OPENAI_API_KEY=sk-...`
2) Install deps: `npm i openai nodemailer`
3) Put your SMTP + ingest emails in `super_chase_agents.manifest.json`.
4) Run: `node index.js`

This template stubs tools and runs bootstrap steps (seed docs + schedule). 
Wire real schedulers/webhooks when ready.
